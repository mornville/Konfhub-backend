# Konfhub-backend

## Installing
* Clone repo using ```https://github.com/mornville/Konfhub-backend.git```
* ```cd Konfhub-backend```
* Set up virtual environment
* Install requirements - ```pip install requirements.txt```

## Running
* ```python main.py```
